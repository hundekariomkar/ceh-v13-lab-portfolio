# CEH v13 Lab Reports 🔐

This repository documents hands-on cybersecurity labs completed during preparation for the Certified Ethical Hacker (CEH v13) certification.

The content is structured module-wise and rewritten in my own words to reflect practical execution, methodology, findings, and mitigation strategies.

⚠️ Disclaimer: This repository does not contain proprietary EC-Council material. All documentation represents personal lab work and learning outcomes.

---

## 📚 Repository Structure

The repository is organized according to CEH v13 modules:

### 01 – Introduction
Foundational ethical hacking concepts, security principles, and attack lifecycle overview.

### 02 – Footprinting 🔎
Google dorking, GHDB analysis, WHOIS lookup, theHarvester, OSINT-based reconnaissance.

### 03 – Scanning Networks
Host discovery, TCP/UDP scanning, service version detection, OS fingerprinting.

### 04 – Enumeration
SMB, SNMP, NetBIOS, user and service enumeration techniques.

### 05 – Vulnerability Analysis
Vulnerability scanning, CVE identification, risk evaluation, and security assessment.

### 06 – System Hacking
Password attacks, privilege escalation, exploitation techniques.

### 07 – Malware Threats
Malware concepts, analysis fundamentals, defensive awareness.

### 08 – Sniffing
Packet capture, protocol analysis, credential interception techniques.

### 09 – Social Engineering
Phishing techniques, human factor vulnerabilities, mitigation strategies.

### 10 – Denial of Service
DoS attack concepts, resource exhaustion techniques, defensive mitigation.

### 11 – Session Hijacking
Session management weaknesses, cookie manipulation, authentication flaws.

### 12 – Evading IDS, Firewalls & Honeypots
Defense evasion concepts and security bypass techniques.

### 13 – Hacking Web Servers
Web server misconfiguration testing and exposure analysis.

### 14 – Hacking Web Applications
Web application security testing aligned with common vulnerability categories.

### 15 – SQL Injection
SQL injection testing techniques and mitigation strategies.

### 16 – Hacking Wireless Networks
Wireless reconnaissance, WPA/WPA2 attack concepts, wireless hardening.
